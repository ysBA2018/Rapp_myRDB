      Auswahl der Organisation und Hochladen der Datei

      Laden Sie die täglich neu erzeugten Berichtsergebnise "BA-Rechte" aus IIQ in den Downloads-Ordner:

      1. Selektieren Sie dazu die Auswertungen -> Berichte -> Berichtsergebnisse -> Suche nach "BA-Rechte"
      2. Vor dem Import müssen die bekannten Schwächen dieser Datei behoben werden, dazu gibt es zwei verschiedene Möglichkeiten:
        1. Nutzen des Konvertierungs-Tools
          1. Bitte laden Sie einmalig die beiden Dateien in Ihren Downloads-Ordner, die unter O:\10_ZIAI\70_A2\04_Intern\TF-Zertifizierung\Tools liegen
          2. Starten Sie in Ihrem Downloads-Ordner die Datei "konvertiere.bat"
        2. Manuelle Konvertierung
          1. Öffnen der Datei mit Notepad++ oder ähnlichem Werkzeug, das regexp versteht (nicht Excel wegen Unicode!)
          2. Ersetzen Sie alle harten Zeilenumbrüche innerhalb von Kommentarfeldern:
            Ersetze alle     \n^([^xv])
            gegen     \1
          3. Ersetzen Sie alle einfachen Backslashes gegen doppelte:
            Ersetze alle     ([^\\])\\([^\\])
            gegen     \1\\\\\2
      3. Wählen Sie die so erzeugte Datei unten über "Durchsuchen..."
