    Datenimport

    Der Import einer CSV-Datei erfolgt in fünf Schritten:

    1. Wählen der Organisation, zu der die Daten hochgeladen werden
    2. Hochladen der Datei zum Server
    3. Bearbeiten neuer User oder von Usern, die ihre Organisation gewechselt haben oder die wieder aktiviert werden
    4. Bestätigen des Löschens nicht mehr erkannter User
    5. Einarbeiten der einzelnen Berechtigungen in die Datenbanken
